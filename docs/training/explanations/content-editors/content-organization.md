# Content Organization

**Type:** Explanation  
**Audience:** Content Editors

## How Content is Organized

### By Type

Content organized by type:
- **Newsletters** - `src/pages/newsletters/`
- **Resources** - `src/pages/content/`
- **Images** - `public/images/`
- **Booklets** - `public/booklets/`

### By Purpose

Each type serves a purpose:
- **Newsletters** - Monthly updates
- **Resources** - Educational content
- **Images** - Visual assets
- **Booklets** - Downloadable materials

## File Naming

### Newsletters

Date-based: `YYYY-MM.md`

### Resources

Descriptive: `kebab-case.md`

### Images

Descriptive: `descriptive-name.png`

## Why This Organization?

- **Easy to Find** - Clear location for each type
- **Consistent** - Predictable file locations
- **Scalable** - Easy to add more content
