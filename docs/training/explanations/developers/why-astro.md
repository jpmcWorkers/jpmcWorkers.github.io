# Why Astro?

**Type:** Explanation  
**Audience:** Developers

## Why Astro Was Chosen

Astro is perfect for this content-heavy website because:

### 1. Fast by Default

- Ships zero JavaScript by default
- Only adds interactivity where needed
- Perfect for content sites

### 2. Content-Focused

- Great for blogs, documentation, content sites
- Built-in content collections
- Markdown support

### 3. Component-Based

- Use any framework (React, Vue, or plain HTML)
- Component islands architecture
- Mix and match as needed

### 4. Developer Experience

- TypeScript support
- File-based routing
- Great tooling

### 5. Static Site Generation

- Perfect for GitHub Pages
- Fast builds
- Secure (no server)

## Benefits for This Project

- **Content Management** - Collections make content type-safe
- **Performance** - Static HTML is fast
- **Simplicity** - No complex build setup
- **Flexibility** - Can add interactivity when needed
