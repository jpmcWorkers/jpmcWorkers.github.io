# Content Types Reference

**Type:** Reference  
**Audience:** Content Editors

## Newsletters

Monthly newsletter updates.

**Location:** `src/pages/newsletters/`
**Format:** `YYYY-MM.md`
**Display:** `/newsletters` page

## Resources

Informational resource pages.

**Location:** `src/pages/content/`
**Format:** `kebab-case.md`
**Display:** `/content` page

## Images

Media library images.

**Location:** `public/images/`
**Format:** Various image formats
**Display:** `/image` page

## Booklets

Printable booklets and PDFs.

**Location:** `public/booklets/`
**Format:** PDF or images in folders
**Display:** `/booklet` page
