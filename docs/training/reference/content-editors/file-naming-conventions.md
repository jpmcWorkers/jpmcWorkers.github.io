# File Naming Conventions Reference

**Type:** Reference  
**Audience:** Content Editors

## Newsletters

**Format:** `YYYY-MM.md`

**Examples:**
- `2025-12.md` - December 2025
- `2026-01.md` - January 2026

**Location:** `src/pages/newsletters/`

## Resources

**Format:** `kebab-case.md`

**Examples:**
- `qa.md`
- `mission.md`
- `rto-petition-status.md`

**Location:** `src/pages/content/`

## Images

**Format:** `descriptive-name.png` (or `.jpg`, `.webp`)

**Examples:**
- `pizza-party-poster.png`
- `newsletter-header-dec-2025.png`

**Location:** `public/images/`

## Booklets

**Format:** Folder with descriptive name, containing PDF/image

**Examples:**
- `right-to-union/Right-to-Union.pdf`
- `organizing-guide/organizing-guide.pdf`

**Location:** `public/booklets/`

## Best Practices

- Use lowercase
- Use hyphens for spaces
- Be descriptive
- Keep names concise
