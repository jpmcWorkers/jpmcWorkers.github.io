# Image Guidelines Reference

**Type:** Reference  
**Audience:** Content Editors

## Supported Formats

- PNG (recommended for graphics)
- JPG/JPEG (for photos)
- WebP (modern format)
- GIF (for animations)

## File Sizes

- Small images: < 100KB
- Medium images: < 500KB
- Large images: < 1MB (when possible)

## Dimensions

- Posters: 1500x750px or similar
- Logos: 200x200px or similar
- Photos: Optimize to reasonable size

## Optimization

Optimize images before uploading:
- Use compression tools
- Choose appropriate format
- Resize if needed

## Alt Text

Always include descriptive alt text:

```markdown
![Description of image content](/images/path.png)
```
