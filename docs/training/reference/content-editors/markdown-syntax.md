# Markdown Syntax Reference

**Type:** Reference  
**Audience:** Content Editors

## Headings

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
```

## Text Formatting

```markdown
**bold** or __bold__
*italic* or _italic_
***bold and italic***
~~strikethrough~~
`inline code`
```

## Lists

### Unordered

```markdown
- Item 1
- Item 2
  - Sub-item
```

### Ordered

```markdown
1. First
2. Second
3. Third
```

## Links

```markdown
[Link text](https://example.com)
[Internal link](/path)
```

## Images

```markdown
![Alt text](/images/path.png)
```

## Code Blocks

````markdown
```language
code here
```
````

## Blockquotes

```markdown
> Quote text
> Multiple lines
```

## Tables

```markdown
| Column 1 | Column 2 |
|----------|----------|
| Data     | Data     |
```

## Horizontal Rules

```markdown
---
```
