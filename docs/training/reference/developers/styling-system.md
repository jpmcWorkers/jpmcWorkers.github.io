# Styling System Reference

**Type:** Reference  
**Audience:** Developers

## Tailwind CSS

This project uses Tailwind CSS v4 for styling.

## JPMC Brand Colors

### Blue (Primary)

- `jpmc-blue-50` through `jpmc-blue-950`
- Primary brand color
- Usage: Primary actions, links, headers

### Red (Secondary)

- `jpmc-red-50` through `jpmc-red-950`
- Secondary brand color
- Usage: Accents, important elements

### Gold (Accent)

- `jpmc-gold-50` through `jpmc-gold-950`
- Accent color
- Usage: Highlights, special elements

## Common Utility Classes

### Spacing

- `p-{size}` - Padding
- `m-{size}` - Margin
- `px-{size}` - Horizontal padding
- `py-{size}` - Vertical padding

### Colors

- `bg-{color}` - Background color
- `text-{color}` - Text color
- `border-{color}` - Border color

### Typography

- `text-{size}` - Font size
- `font-{weight}` - Font weight
- `text-{align}` - Text alignment

### Layout

- `flex` - Flexbox
- `grid` - Grid layout
- `container` - Container
- `mx-auto` - Center horizontally

### Responsive

- `md:` - Tablet and up
- `lg:` - Desktop and up
- `xl:` - Large desktop and up

## Custom Classes

Defined in `src/styles/global.css`:
- JPMC brand colors
- Custom theme configuration
